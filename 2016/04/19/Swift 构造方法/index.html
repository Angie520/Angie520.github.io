Swift 构造方法

2016.04.19 Angie Swift  热度 ℃

定一个对象类。。Person。。编写相关的构造方法

class Person: NSObject {

//如果定义属性的时候没有初始化，那么必须在后面写上？
//Swift要求，属性必须有初始值
//只要在构造方法中对属性进行了初始化，那么不用写？
var name: String?
//如果是定义一个对象属性，那么后面可以写上？
//如果是定义一个基本数据类型属性，那么建议直接赋值为0
//因为super.init()方法在分配存储空间的时候，如果发现属性是一个对象，并且是一个可选类型，那么会给这个属性分配存储空间，但是如果属性是基本数据类型，并且是可选类型，那么super.init()不会给该属性分配存储空间
// var age: Int?—错误的定义

var age:Int = 0
//重写父类默认的构造方法
//注意点：如果自定义了构造方法，并且没有重写父类默认的构造方法
//那么默认的构造方法就会失效。。下面即是父类默认的构造方法
override init() {
self.name = “小明”
self.age = 30
}
//自定义构造方法
//Swift中有方法重载的概念
//允许有同名的方法，只要形参或返回值不一样
init(name:String,age:Int) {
self.name = name
self.age = age
}
init(dict:[String: NSObject]) {
//注意点：如果想在构造方法中使用KVC给属性赋值
//必须在使用KVC之前必须调用super.init()
//调用super.init()的目的就是为了能在KVC赋值前给属性分配存储空间。。这个时候属性必须加？
super.init()
setValuesForKeysWithDictionary(dict)
}
}
